Setup Instructions
  1. Open Google Cloud Console, create or select a project, and enable the Google Sheets API.
  2. Under IAM & Admin → Service Accounts, create a service account. Generate a JSON key and save it somewhere safe on the server, e.g. /etc/lastwar/service-account.json.
  3. Open the target Google Sheet, click Share, and grant Editor access to the service account email (the "client_email" field inside the JSON key).
  4. Copy the Spreadsheet ID from the sheet URL:
    https://docs.google.com/spreadsheets/d/<SPREADSHEET_ID>/edit
  5. Set the following environment variables before starting the app: 
    GOOGLE_SHEETS_ENABLED=true
GOOGLE_SHEETS_CREDENTIALS_FILE=/etc/lastwar/service-account.json
GOOGLE_SHEETS_SPREADSHEET_ID=<your_spreadsheet_id>
GOOGLE_SHEETS_SHEET_NAME=Rankings    # optional, default is Rankings
  6. Restart the app, return to this page, and click Test Connection to verify everything works.
  7. Use Export to Google Sheets (here or on the Data page) to push all ranking entries. The sheet tab is fully rewritten on each export.
